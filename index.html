Les projets

J'ai pu réaliser différents projets avec différents langages en équipe ou seul.

Projet seul : Reconnaissance d'image avec NodeJS

Mettre en place un outil de reconnaissance d'image, avec NodeJS, pour l'intégration à un robot dans le cadre du projet VITAAL (assistance à la personne).
Réalisé en M1 à l'UBO sur 2 mois.

Projet seul : Récupération de données de ruchers et affichage graphique avec Java

Coder une application Java permettant d'afficher, avec des graphiques, les données de différents ruchers.
Réalisé en L3 en distanciel sur 2 mois.

Projet en équipe : Réalisation dune application de badgeage pour porte automatique

Proposer une application de badgeage à l'agent d'accueil du lycée avec différentes possibilités sur interface web et une application serveur codée en C++. Les cartes d'accès utilisaient du RFID.
Réalisé en 2ème année de BTS sur plusieurs semaines.

Stage : Mise en place de l'outil d'affichage dynamique Xibo

Déployer l'outil Xibo dans les différents batiments de l'Institut PPrime. Le tester sur différents OS du côté client et trouver une version stable du côté serveur. Rédaction de différents tutoriels pour les futurs utilisateurs.
Réalisé en 1ère année de BTS sur 2 mois.
